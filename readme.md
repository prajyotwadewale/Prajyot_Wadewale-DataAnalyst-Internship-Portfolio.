# 📂 Tasks Overview

---

## 🔹 Task 1: Data Immersion & Wrangling

### 🎯 Objective

To understand, clean, and prepare raw data for analysis.

### 🛠️ Key Work

* Data loading and exploration
* Handling missing values
* Removing duplicates
* Cleaning inconsistent data
* Feature engineering

### 📊 Output

* Cleaned dataset
* Data dictionary
* Python cleaning script

---

## 🔹 Task 2: Exploratory Data Analysis (EDA) & Business Intelligence

### 🎯 Objective

To uncover patterns, trends, and insights from the dataset.

### 🛠️ Key Work

* Descriptive statistics
* Univariate & bivariate analysis
* Data visualization (charts & graphs)
* SQL queries for business questions

### 📊 Output

* EDA report
* SQL queries
* Business insights

---

## 🔹 Task 3: Deep-Dive Analysis & Interactive Dashboard

### 🎯 Objective

To solve complex business problems and build dashboards.

### 🛠️ Key Work

* KPI definition (e.g., revenue, churn rate)
* Deep-dive analysis (cohort / segmentation / funnel)
* Dashboard creation using BI tools

### 📊 Output

* Interactive dashboard
* KPI report

---

## 🔹 Task 4: Data Storytelling & Statistical Validation

### 🎯 Objective

To present insights in a business-friendly way and validate findings.

### 🛠️ Key Work

* Creating a data story
* Building presentation slides
* Hypothesis testing (T-test / Chi-square)
* Drawing conclusions

### 📊 Output

* Final presentation deck
* Statistical analysis report

### 📊 Output
