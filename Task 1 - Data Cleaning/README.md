## 🎯 Objective

The objective of this task is to clean and preprocess raw e-commerce data to make it suitable for analysis by handling missing values, duplicates, and incorrect data formats.

---

## 📂 Dataset

* Dataset: E-commerce Dataset
* Source: Kaggle (Online Retail Dataset)

---

## 🔧 Data Cleaning Steps

* Removed missing values
* Dropped duplicate records
* Converted data types (Date, Numeric fields)
* Created new column: **TotalPrice = Quantity × UnitPrice**
* Filtered invalid values (negative or zero)

---

## 📊 Output

* Cleaned dataset ready for analysis
* Improved data quality and consistency
